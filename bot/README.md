# Beauty Studio Telegram Bot with Web App

This project contains a Telegram bot for managing a beauty studio schedule with an integrated web application.

## Features
- Telegram bot interface
- Web App integration
- Simple scheduling system

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up environment variables in `.env`:
- `TELEGRAM_BOT_TOKEN` - Your Telegram bot token
- `WEB_APP_URL` - URL where your web app is hosted
- `BACKEND_URL` - Backend API URL (if applicable)

3. Run the bot:
```bash
python bot.py
```

## Deployment
This application is configured for deployment on Railway.app
